Const debounce

  • debounce<A>(f: Function<A, void>, delay: Milliseconds): Function<A, void>
    • Defined in lib/control/timer.ts:20

    debounce delays the application of a function until the specified time has passed.

    If multiple attempts to apply the function have occured, then each attempt will restart the delay process. The function will only ever be applied once after the delay, using the value of the final attempt for application.

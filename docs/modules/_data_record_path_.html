External module "data/record/path"

This module provides a syntax and associated functions for getting and setting values on ES objects easily.

Given a path, a value can either be retrieved or set on an object.

The path syntax follows typical ES dot notation, bracket notation or a mixture of both.

Note that quotes are not used when describing a path via bracket notation.

If you need to use a dot or square brackets in your paths, prefix them with the "" (backslash) character.

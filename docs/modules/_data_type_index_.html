Const test

  • test<V, T>(value: V, t: Pattern<T>): boolean
    • Defined in lib/data/type/index.ts:90

    test whether a value conforms to some pattern.

    This function is made available mainly for a crude pattern matching machinery that works as followss: string -> Matches on the value of the string. number -> Matches on the value of the number. boolean -> Matches on the value of the boolean. object -> Each key of the object is matched on the value, all must match. function -> Treated as a constructor and results in an instanceof check or for String,Number and Boolean, this uses the typeof check. If the function is RegExp then we uses the RegExp.test function instead.
